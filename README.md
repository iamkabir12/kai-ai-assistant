# 🤖 KAI - AI Voice Assistant

KAI is a Python-based AI Voice Assistant that can perform various tasks using voice commands. It includes a GUI interface and supports system-level operations.

---

## 🚀 Features

* 🎤 Voice Command Recognition
* 🖥️ GUI Interface (User-friendly)
* 🌐 Open websites (Google, YouTube, etc.)
* 📂 Open applications
* 🔊 Control system functions (volume, etc.)
* 📸 Camera access
* 🔍 Perform web searches
* ⏻ Shutdown / Restart system

---

## 🛠️ Technologies Used

* Python 🐍
* SpeechRecognition
* pyttsx3 (Text-to-Speech)
* Tkinter (GUI)
* OS & System Libraries

---

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/iamkabir12/kai-ai-assistant.git
```

2. Navigate to project folder:

```
cd kai-ai-assistant
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the assistant:

```
python main.py
```

Click **Start Listening** and give voice commands.

---

## 🧠 Example Commands

* "Open YouTube"
* "Search Python tutorials"
* "Open camera"
* "Shutdown system"
